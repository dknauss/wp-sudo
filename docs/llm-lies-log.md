LLM Lies Log
=============
Fabricated, outdated, and unverified claims generated by LLMs and
committed to repositories without verification against live sources.
This file exists for prevarication verification and prevention.

Audited: 2026-03-09 (v2.14 milestone audit — 4 new entries found)
Previous: 2026-03-03 (v2.10.2 release — no new fabrications found)
Repos:   wp-sudo, wordpress-2fa-ecosystem

ROOT CAUSE
----------
All errors stem from the same pattern: generating reference content from
training data instead of verifying against live, authoritative sources.
The WordPress.org Plugin Info API, WordPress.org SVN, and GitHub raw
file URLs were all available and could have been queried in seconds.

The fabricated method and meta key names are the worst category — these
are not stale facts but invented strings that sound plausible. A human
reading the documentation would have no reason to doubt them, and code
using them would fail silently or throw errors.
